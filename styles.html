This section is about formatting text.

This text is bold. To bold text, surround the text you want to bold within a paragraph with "b" tags.

This text is italic. To italicize text, surround the text you want to italicize within a paragraph with "i" tags.

This is subscript and superscript. To get subscript, surround the text with "sub" tags. To get superscript, surround the text with "sup" tags.

This is highlighted text. To highlight text, surround it with "mark" tags.

This is deleted text. To strikethrough text, surround it with "del" tags.

This is underlined text. To underline text, surround it with "ins" tags.



If you want to keep a certain formatting, you can't simply use "p" tags. You have to use "pre" tags.
